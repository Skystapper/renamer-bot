### Deploy in your vps
```sh
git clone https://github.com/prgofficial/RenameBot-PermTB
cd RenameBot-PermTB
virtualenv -p python3 VENV
. ./VENV/bin/activate
pip install -r requirements.txt
# <Create config.py appropriately>
python3 -m tobrot
```
